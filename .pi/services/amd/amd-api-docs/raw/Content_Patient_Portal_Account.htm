Look Up Patient Portal Account

Example JSON:
Copy 
{
  "ppmdmsg": {
    "@action": "lookuppatientportalaccount",
    "@class": "lookup",
    "@msgtime": "9/13/2024 8:49:35 AM",
    "@exactmatch": "1",
    "@fullname": "PATIENT,PORTAL",
    "@orderby": "fullname",
    "@page": "1"
  }
}
