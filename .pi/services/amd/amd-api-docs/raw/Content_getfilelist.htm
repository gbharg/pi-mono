Getfilelist Message Specification

"Getfilelist" Action Message Entity Layout

Copy 
<ppmdmsg>
    <filelist>
        <file/>
    </filelist>
    <categorylist>
        <category/>
    </categorylist>
    <typelist>
        <type/>
    </typelist>
</ppmdmsg>

Note: The "getfilelist" returns the same results as the "uploadfile" action.

"Getfilelist" Action Message XML Schema Entity Definition

Element Name: ppmdmsg

Description

This is the root element; it describes the action and class that will handle the message.
