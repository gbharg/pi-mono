Charge Rebill

POST api/claim/chargerebill

Creates a charge rebill for a claim. This endpoint allows you to update charge details and optionally add notes, resubmission codes, and action history.

HTTP Status Codes

Code

Description

200

Ok

400

Bad request
Request Body
Copy 
{
  "chargeDetailFID": 2, // required, min value: 2, source: vw_ODBC_actv_ChargeDetail
  "sequenceNumber": 1, // required, min value: 1
  "chargeOnly": true,
  "rebillNote": "string",
  "icn": "string",
  "actionHistory": {
    "actionFID": 1, // required, min value: 1
    "nextActionFID": 1, // nullable, min value: 1
    "nextActionDate": "2025-12-15T00:00:00Z", // nullable 
    "note": "string"
  },
  "isRemoveForceWorklist": true,
  "resubmissionCode": "string"
}

 

Request Parameters

Parameter Type Required Description
chargeDetailFID Integer Yes Unique ID for the charge detail. Minimum value: 2.
sequenceNumber Integer Yes Sequence number for the charge. Minimum value: 1.
chargeOnly Boolean No If true, only the charge will be rebilled.
rebillNote String No Additional note for the rebill.
icn String No Internal Control Number.
actionHistory.actionFID Integer Yes Action ID. Minimum value: 1.
actionHistory.nextActionFID Integer No Next action ID. Minimum value: 1.
actionHistory.nextActionDate String No Date for next action in ISO 8601 format.
actionHistory.note String No Note for the action history.
isRemoveForceWorklist Boolean No Indicates if forced worklist should be removed.
resubmissionCode String No Code for resubmission.

 

Response Body
Copy 
{
  "sortdate": "2025-12-15T00:00:00Z",
  "actionDate": "string",
  "name": "string",
  "note": "string",
  "nextAction": "string",
  "followUpDate": "string",
  "createdBy": "string",
  "type": "string"
}

 

Response Fields

Field Type Description
sortdate String Date used for sorting (ISO 8601 format).
actionDate String Date of the action.
name String Name associated with the action.
note String Note related to the action.
nextAction String Description of the next action.
followUpDate String Date for follow-up.
createdBy String User who created the action.
type String Type of action.
