Get Visits By ID

Action: getvisitsbyid

Requirements: At least one visit id must be present in the payload within the visitlist object.

Sample XML:
Copy 
<ppmdmsg action="getvisitsbyid" class="api">
    <visitlist>
        <visit id="123"/>
    </visitlist> 
    ...
</ppmdmsg>
Sample JSON:
Copy 
{
    "ppmdmsg": {       
        "@action": "getvisitsbyid",
        "@class": "api",
        "visitlist": {
            "visit": [{
                    "@id": "123"
                }]
        }
        …
    }
}
