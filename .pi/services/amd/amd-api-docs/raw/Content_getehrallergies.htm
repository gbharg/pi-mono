Get an Allergies List

Action:getehrallergies

Sample XML:
Copy 
<ppmdmsg
  action="getehrallergies"
  class="api"
  msgtime="6/7/2016 6:28:09 PM"
  patientid="15963"
>
  <allergy
    allergy="Allergy"
    updatedate="UpdateDate"
    reaction="Reaction"
    treatment="Treatment"
    username="UserName"
    drugallergycatid="DrugAllergyCatID"
    ncallergycatid="NcAllergyCatId"
    status="Status"
    StartDate="StartDate"
  />
</ppmdmsg>;
Sample JSON:
Copy 
{ 
  "ppmdmsg": { 
    "@action": "getehrallergies", 
    "@class": "api", 
    "@msgtime": "6/7/2016 6:28:09 PM", 
    "@patientid": "15963", 
    "allergy": { 
      "@allergy": "Allergy", 
      "@updatedate": "UpdateDate", 
      "@reaction": "Reaction", 
      "@treatment": "Treatment", 
      "@username": "UserName", 
      "@drugallergycatid": "DrugAllergyCatID", 
      "@ncallergycatid": "NcAllergyCatId", 
      "@status": "Status", 
      "@StartDate": "StartDate" 
    } 
  } 
}

Attributes:

patientid: The unique ID of the patient whose allergies you wish to retrieve

Results:

This request returns a < allergylist> element or object that contains a list of all allergies specific for the certain patient. It will return all allergies current and historic. Every returned element (<allergy >) will contain an ID attribute.
