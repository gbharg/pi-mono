Mass Rebill

POST api/claim/massrebill

This endpoint allows you to perform a mass rebill of claims based on specified visit information and search criteria.

HTTP Status Codes

Code

Description

200

Ok

400

Bad request
Request Body
Copy 
{
  "visitInfo": [ // required.
    {
      "visitFID": 2, // required, min value: 2
      "chargeDetailIds": "string", // required.
      "icn": "string"
    }
  ],
  "rebillNote": "string",
  "actionHistory": {
    "actionFID": 1, // required, min value: 1
    "nextActionFID": 1, // nullable, min value: 1
    "nextActionDate": "2025-12-15T00:00:00Z", // nullable.
    "note": "string"
  },
  "isRemoveForceWorklist": false,
  "resubCode": "string",
  "correctedClaim": {
    "includeOnPaperClaim": false,
    "visitNote": "string"
  },
  "messageTime": "2025-12-15T00:00:00Z", // nullable.
  "searchCriteria": {
    "dos": {
      "from": "2025-12-15T00:00:00Z", // required.
      "to": "2025-12-15T00:00:00Z" // required.
    },
    "patients": ["string"],
    "carriers": ["string"],
    "paymentReasons": ["string"],
    "billingDate": {
      "from": "2025-12-15T00:00:00Z", // required.
      "to": "2025-12-15T00:00:00Z" // required.
    },
    "carrierCategories": ["string"],
    "denialCategories": ["string"],
    "visitIds": ["string"],
    "providers": ["string"],
    "facilities": ["string"],
    "chargeCodes": ["string"],
    "claimForms": ["string"],
    "claimRuns": ["string"],
    "submitterTypes": ["string"],
    "minimumClaimBalance": 0.00, // nullable.
    "financialClasses": ["string"],
    "chargeCodeCategories": ["string"],
    "agingDayRange": 0, // nullable.
    "cpid": "string",
    "npid": "string",
    "carrierWSID": "string"
  }
}

Important Note: At least one visit must be included in the visitInfo array; otherwise, the request will return 400 Bad Request.

Request Parameters

Parameter Type Required Description
visitInfo Array Yes List of visits to rebill. Must contain at least one object.
visitFID Integer Yes Unique visit identifier. Minimum value: 2.
chargeDetailIds String Yes IDs of charge details associated with the visit.
icn String No Internal Control Number for the claim.
rebillNote String No Note explaining the reason for rebill.
actionHistory Object Yes Details of the action performed and next steps.
actionFID Integer Yes Action identifier. Minimum value: 1.
nextActionFID Integer No Identifier for the next action. Minimum value: 1.
nextActionDate String (ISO) No Date for the next action in ISO 8601 format.
note String No Additional notes for the action.
isRemoveForceWorklist Boolean No Indicates if forced worklist should be removed.
resubCode String No Resubmission code for the claim.
correctedClaim Object No Details for corrected claim submission.
includeOnPaperClaim Boolean No Whether to include corrected claim on paper submission.
visitNote String No Note for the corrected claim.
messageTime String (ISO) No Timestamp of the message in ISO 8601 format.
searchCriteria Object No Filters for selecting claims to rebill.
dos.from String (ISO) Yes Start date of service.
dos.to String (ISO) Yes End date of service.
patients Array No List of patient names or IDs.
carriers Array No List of carrier names or IDs.
paymentReasons Array No Payment reason codes.
billingDate.from String (ISO) Yes Start of billing date range.
billingDate.to String (ISO) Yes End of billing date range.
carrierCategories Array No Carrier category filters.
denialCategories Array No Denial category filters.
visitIds Array No Specific visit IDs to include.
providers Array No Provider identifiers.
facilities Array No Facility identifiers.
chargeCodes Array No Charge codes to filter.
claimForms Array No Claim form types.
claimRuns Array No Claim run identifiers.
submitterTypes Array No Submitter type filters.
minimumClaimBalance Decimal No Minimum claim balance to include.
financialClasses Array No Financial class filters.
chargeCodeCategories Array No Charge code category filters.
agingDayRange Integer No Aging day range filter.
cpid String No CPID identifier.
npid String No NPID identifier.
carrierWSID String No Carrier WSID identifier.

 

Response Body
Copy 
{
  "affectedVisitsCount": 0,
  "reportUrl": "string"
}

 

Response Parameters

Parameter Type Description
affectedVisitsCount Integer Number of visits affected by the rebill operation.
reportUrl String URL to download the generated report.
