System Defaults Requests

Getting System Defaults

Action:getsysdefaults

Sample XML:
Copy 
<ppmdmsg action="getsysdefaults" class="api" msgtime="5/22/2015 3:56:48 PM"/>
Sample JSON:
Copy 
{ 
  "ppmdmsg": { 
    "@action": "getsysdefaults", 
    "@class": "api", 
    "@msgtime": "5/22/2015 3:56:48 PM" 
  } 
}
