# End of Day Checklist

Run this before closing out for the day. Every item must be true.

## Files & Sync
- [ ] All tasks in todo.md have matching Linear issues with IDs
- [ ] Linear issues reflect current status (todo/in-progress/done)
- [ ] All decisions saved to project decisions.md (categorized, timestamped, session-referenced)
- [ ] All learnings saved to memory/learnings.md (categorized)
- [ ] changelog.md updated with any infrastructure/config changes made today
- [ ] All changes committed and pushed to project branch

## Session Close
- [ ] Session summary written (three layers: status, narrative, commitments)
- [ ] Summary passes the test: "if Gautam asked what we did next week, could I answer from this?"
- [ ] state.md and context.md updated for next session bootstrap
- [ ] todo.md updated — new items added, completed items checked off, dates current
- [ ] No orphaned follow-ups — everything promised is tracked somewhere

## Quality
- [ ] Did I verify before acting? (checked docs, validated paths, confirmed assumptions)
- [ ] Did I stay on one topic at a time?
- [ ] Did I avoid batching questions?
- [ ] Did I keep Gautam informed proactively?
- [ ] Did anything go in .pi/ that doesn't belong there?

## Profile & Memory
- [ ] Any new preferences or corrections from Gautam saved to gautam.md or preferences.md
- [ ] Any new learnings saved immediately, not deferred
- [ ] pi.md updated if my role or growth areas evolved
