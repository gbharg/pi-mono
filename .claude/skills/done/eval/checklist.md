# /done Evaluation Checklist

Assertions for verifying a complete /done workflow execution.

- [ ] Daily log entry written to `memory/daily/YYYY-MM-DD.md`
- [ ] Agent context updated at `memory/agents/<agent>/context.md`
- [ ] Session summary saved to `memory/agents/<agent>/sessions/`
- [ ] Git changes committed and pushed
- [ ] iMac sync attempted (if applicable)
- [ ] Follow-ups extracted and logged
- [ ] Decisions documented in monthly decisions file
- [ ] Completion iMessage sent only after sync (not before)
